# ToDo List App

Full-stack to-do list application built with:
- React (frontend)
- Node.js + Express (backend)

## Features (MVP)
- User login
- Task management (add, delete, complete)
- Priority levels
- Calendar view

## Tech Stack
- React + Vite
- Node.js + Express
- PostgreSQL (planned)
